To get started,
   cd to this directory (server)

Install pipenv on your machine
   >> pip install pipenv

Setup virtual env and activate it
    >> pipenv shell

Install all dependencies
    >> pipenv install

Run the flask app
    >> python app.py

    the server should start on localhost:5000 by default.
    check your terminal :)
